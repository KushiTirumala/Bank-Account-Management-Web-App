Bank Account Management Web App (HTML, CSS, JavaScript)

A dynamic web application that allows users to create a bank account, deposit, withdraw, and check balance using JavaScript DOM manipulation and event handling.

Project Overview

This project demonstrates front-end web development with interactive functionality. Users can manage a bank account in the browser without page reloads. It showcases JavaScript classes, DOM updates, event listeners, and dynamic UI changes.

Features

Create a bank account with account number and holder name

Deposit money dynamically

Withdraw money with validation for insufficient balance

Real-time balance display

Interactive messages for user actions

Technologies Used

HTML5 – Page structure

CSS3 – Styling and responsive layout

JavaScript – DOM manipulation, event handling, OOP
